# 🏋️ PRODIGY FORGE - Gym Membership Management

A full-stack web application built with **Django** and **MySQL** to manage gym memberships efficiently.

## 🚀 Features
- **Dashboard:** View all active members and their plan details.
- **Member Management:** Add new members and delete expired accounts.
- **Upgrade Logic:** Members can upgrade to higher tiers (Pro/Elite) but cannot downgrade.
- **Automated Expiry:** Registration and Expiry dates are calculated automatically based on duration.
- **Clean UI:** Responsive design using Bootstrap 5.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, Bootstrap 5

## 🔧 Installation
1. Clone the repo: `git clone https://github.com/yourusername/gym-management.git`
2. Install requirements: `pip install -r requirements.txt`
3. Configure MySQL in `settings.py`.
4. Run migrations: `python manage.py migrate`
5. Start server: `python manage.py runserver`
